What is HTML

Hypertext Markup Language (HTML) is a markup language used in documents that typically contain direct links (URLs) that make up the 'world wide web'.

Current versions

HTML Version 5 is now being used, having been recommended by the World Wide Web Consortium (W3C) in November 2014.

It was a long time coming, superseding version 4 which got W3C recommendation in 1999.

What it consists of

HTML is a formatting language, that uses elements, comprising of tags, to impose formal structure.

Tags are of different types, e.g.

  • some tags do not apply to particular document text, and are 'void tags', with no end tag - such as br and img
  • other elements do have end tags - e.g. p, head, body, html etc
  • 'inline' elements impose their effects on text within that text structure e.g. span, or br
  • 'block' elements create an independent 'block' effect within a page e.g. form, div

HTML uses CSS

HTML works in tandem with 'cascading style sheets' (CSS), to provide a more orderly way to style webpages.

An HTML file links to a CSS file its using, or CSS code can be directly embedded via style tags.

This is particularly important for maintaining styling consistency in pages of large websites. HTML currently uses CSS version 3.
